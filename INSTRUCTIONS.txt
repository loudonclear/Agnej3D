
Controls:
 - Press ESC to quit.
 - F to toggle fullscreen.
 - Move with WASD and use the mouse to pan.
 - Space to start / create shapes (one turn).
 - Mouse wheel changes distance.
 - Single left-click picks up/puts down a shape (one turn).
 - Right click hold and release throws a shape (one turn).
 		- Mouse wheel while holding changes throw force.
 - Middle click places a static shape only if it is touching another static
 		shape (three turns)


Stack as high as possible!
