# AGNEJ

A 3D OpenGL/C++ implementation of the Gilbert-Johnson-Keerthi and Expanding Polytope algorithms for arbitrary convex hulls.

![agnej](https://github.com/loudonclear/Agnej3D/blob/master/agnej.png)

# Features

- Full GJK/EPA
- Persistent Contacts
- Collision sleeping
- Stable stacking
- FXAA
